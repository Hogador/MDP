# Google Play Submission Checklist

## Pre-Submission
- [ ] App bundle built (release)
- [ ] ProGuard/R8 rules verified
- [ ] Privacy policy URL
- [ ] Content rating questionnaire
- [ ] Data safety section
- [ ] Crypto compliance (no direct purchase, utility token only)

## Store Listing
- [ ] Title: MDAOPay — Crypto Wallet
- [ ] Description: Gasless crypto wallet with social recovery
- [ ] Feature graphic
- [ ] Screenshots (phone + tablet)
- [ ] Short description (80 chars)

## Technical Requirements
- [ ] Target SDK 34
- [ ] 64-bit support
- [ ] No malware/cheats
- [ ] No misleading claims

## Submission Flow
1. Internal testing (team)
2. Closed testing (alpha)
3. Open testing (beta)
4. Production (gradual rollout: 10% → 50% → 100%)
