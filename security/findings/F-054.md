**Impact:**
HIGH — перебор паролей, account takeover.

**Verification recipe:**

Добавить rate limiting на /auth/login (max 5/min/IP), /auth/register (max 3/min/IP), /auth/refresh (max 10/min/IP). Использовать Redis-based sliding window.
