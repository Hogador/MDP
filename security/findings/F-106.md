### F-106 [MEDIUM] CertificatePinner placeholder хэши

| Field | Value |
|-------|-------|
| **ID** | F-106 |
| **Severity** | MEDIUM |
| **Category** | G3 (networking) |
| **Bug pattern** | `cert-pinner-placeholder` |
| **Fingerprint** | `sha256("G3:app/.../NetworkModule.kt:CertificatePinner:cert-pinner-placeholder")` → `a106000000000000000000000000000000000000000000000000000000000001` |
| **Status** | NEW |
| **Found** | External Cloud Audit 2026-06-29 |

**Description:**
CertificatePinner в NetworkModule.kt содержит placeholder хэши `sha256/AAAA...=`. Сломает HTTPS при первом релизе.

**Impact:**
MEDIUM — любой релиз с pinning'ом недоступен для сети.

**Verification recipe:**
```bash
grep "sha256/[^A]" app/**/NetworkModule.kt
# PASS: реальные SPKI хэши
```

**Required fix:**
Вычислить реальные SPKI pins через openssl, добавить через BuildConfig per-flavor.
