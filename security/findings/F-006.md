**Impact:** Trusted signer signature replay via malleable signature.

```bash
# PASS: "import {ECDSA}" line present + "ECDSA.recover(" used + NO raw "ecrecover("
```

**Required fix:**
```solidity
import {ECDSA} from "@openzeppelin/contracts/utils/cryptography/ECDSA.sol";
```
